# Script ElevenLabs — StudyHub Full (giọng nam · anh / các em)

**Mục đích:** Paste vào ElevenLabs (multilingual v2).  
**Video:** `StudyHubFull`  
**Từ gốc:** StudyHub, Zalo, Drive, QR, HSA, V-ACT, TSA, luyende.id.vn, 2k9, online…  
**Không đọc số điện thoại.**

---

## Cách dùng

1. Voice nam · Stability ~0.45–0.55 · Similarity ~0.75  
2. Tách **12 file** `s01.mp3`…`s12.mp3` → `remotion/public/voice/`  
3. `npm run render:full`  
4. Break: giữ `<break time="…" />` nếu model hỗ trợ; không thì xóa, giữ `...`
